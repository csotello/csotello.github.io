CuteDogStash

Blog site written in Rust using the Yew framework(similar to React.js) and Trunk to bundle generated .wasm and .js files. Aimed toward users on a shared computer, the site allows people to create accounts and post images(specifically of their dog) which others can rate. Storage persists through local storage although some initial values have been provided.
